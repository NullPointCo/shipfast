## Stack
- Node.js + Express
- PostgreSQL (optional) — falls back to an in-memory store for local dev/tests
- Stripe payments (Checkout subscriptions + webhook)
- JWT auth (bcrypt password hashing)
- Docker + Compose
- Render deploy config
- Jest tests

## Quick Start

```bash
npm install
cp .env.example .env        # fill in SECRET_KEY, Stripe keys
npm run dev                 # http://localhost:8080
```

With Docker (uses Postgres):

```bash
docker compose up --build
```

## API
| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | `/api/v1/health` | – | Health check |
| POST | `/api/auth/register` | – | Create account, returns JWT |
| POST | `/api/auth/login` | – | Login, returns JWT |
| GET | `/api/auth/me` | ✅ | Current user |
| POST | `/api/stripe/checkout` | ✅ | Create Stripe Checkout session |
| POST | `/api/stripe/webhook` | 🔒 | Stripe webhook (raw body) |

## Stripe
1. Set `STRIPE_SECRET_KEY` and `STRIPE_WEBHOOK_SECRET` in `.env`.
2. Replace the `price_XXXX` / `price_YYYY` placeholder price IDs in `src/routes/stripe.js` with your real Stripe price IDs.
3. Expose the webhook locally with the Stripe CLI: `stripe listen --forward-to localhost:8080/api/stripe/webhook`.
