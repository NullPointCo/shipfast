# {{PROJECT_NAME}} — Flask + Stripe SaaS Boilerplate

Production-ready Flask 3 starter with auth, a database, and Stripe subscription
billing. Generated by [shipfast](https://github.com/NullPointCo/shipfast).

## Stack
- **Flask 3** — web framework
- **Flask-SQLAlchemy** — ORM (PostgreSQL in prod, SQLite locally)
- **Flask-JWT-Extended** — stateless auth
- **Flask-Bcrypt** — password hashing
- **Stripe** — subscription Checkout + signed webhooks
- **Docker + docker-compose** — local dev
- **Render** — one-click deploy (`render.yaml`)

## Quickstart
```bash
pip install -r requirements.txt
cp .env.example .env   # fill in STRIPE_SECRET_KEY / JWT_SECRET_KEY
python wsgi.py         # or: gunicorn wsgi:app --bind 0.0.0.0:8000
```

## Local dev with Docker
```bash
docker compose up --build
```

## API
| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | `/` | – | Service health/meta |
| GET | `/api/health` | – | Liveness probe |
| POST | `/api/register` | – | Register `{email, password}` → `{access_token, user}` |
| POST | `/api/login` | – | Login → `{access_token, user}` |
| POST | `/api/checkout` | ✅ | Create Stripe Checkout session `{price_id}` |
| POST | `/api/webhook` | – | Stripe webhook (updates subscription status) |
| GET | `/api/profile` | ✅ | Current user |

## Tests
```bash
pytest
```

## Deploy
Push to Render and connect `render.yaml`, or run the Dockerfile anywhere.
Set `DATABASE_URL`, `STRIPE_SECRET_KEY`, `STRIPE_WEBHOOK_SECRET`, `JWT_SECRET_KEY`.
