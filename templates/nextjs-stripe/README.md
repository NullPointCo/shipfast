## Quick start

```bash
npm install
cp .env.example .env.local   # fill in your Stripe test keys + price IDs
npm run dev                  # http://localhost:3000
```

## Stripe setup

1. Create two **recurring** Prices in the Stripe Dashboard.
2. Copy their IDs into `.env.local`:
   - `NEXT_PUBLIC_STRIPE_STARTER_PRICE_ID`
   - `NEXT_PUBLIC_STRIPE_PRO_PRICE_ID`
3. For local webhooks:
   ```bash
   stripe listen --forward-to localhost:3000/api/webhooks
   ```
   Copy the printed `whsec_...` into `STRIPE_WEBHOOK_SECRET`.

## Deploy

```bash
docker build -t {{PROJECT_NAME}} .
docker run -p 3000:3000 --env-file .env.local {{PROJECT_NAME}}
```

Set the same env vars on your host (Render/Fly). Point `NEXT_PUBLIC_BASE_URL` at your live domain.
