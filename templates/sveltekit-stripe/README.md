## Quick start

```bash
npm install
cp .env.example .env          # fill in your Stripe test keys + price IDs
npm run dev                   # http://localhost:5173
```

## Stripe setup

1. Create two **recurring** Prices in the Stripe Dashboard.
2. Copy their IDs into `.env`:
   - `PUBLIC_STRIPE_STARTER_PRICE_ID`
   - `PUBLIC_STRIPE_PRO_PRICE_ID`
3. For local webhooks:
   ```bash
   stripe listen --forward-to localhost:5173/api/webhooks
   ```
   Copy the printed `whsec_...` into `PRIVATE_STRIPE_WEBHOOK_SECRET`.

## Deploy

```bash
docker build -t {{PROJECT_NAME}} .
docker run -p 3000:3000 --env-file .env {{PROJECT_NAME}}
```

Set the same env vars on your host (Render/Fly). Point `PUBLIC_BASE_URL` at your live domain.
