## Setup
```bash
npm install
cp .env.example .env   # fill in your Stripe keys + Price ID
npm run dev
```

## Configure Stripe
1. Create a **Product** + **Price** (recurring) in the Stripe dashboard.
2. Copy the Price ID into `.env` as `STRIPE_PRICE_ID_PRO`.
3. For local webhooks, run `stripe listen --forward-to localhost:5173/api/webhooks` and copy the signing secret into `STRIPE_WEBHOOK_SECRET`.
