## Why LemonSqueezy?
LemonSqueezy acts as the merchant of record, handling global VAT/sales tax and
payouts. You can start selling SaaS subscriptions without a registered company
or a Stripe onboarding review — ideal for indie hackers.

## Quick Start

```bash
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env   # fill in your LemonSqueezy keys
docker compose up db -d
uvicorn app.main:app --reload --port 8080
```

## Endpoints
| Method | Path | Description |
|--------|------|-------------|
| POST | `/api/v1/register` | Create account |
| POST | `/api/v1/login` | Get JWT |
| POST | `/api/v1/checkout` | Create LemonSqueezy checkout |
| POST | `/api/v1/webhooks/lemonsqueezy` | Webhook (HMAC-verified) |

## Webhook setup
In the LemonSqueezy dashboard, point the webhook at
`https://your-domain/api/v1/webhooks/lemonsqueezy` and copy the signing secret
into `LEMONSQUEEZY_WEBHOOK_SECRET`. The handler auto-upgrades a user's `plan`
to `pro` on `subscription_created` / `subscription_updated`.
