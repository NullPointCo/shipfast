## Stack
- FastAPI + Uvicorn
- Supabase (Postgres + Auth + Storage)
- Stripe subscriptions
- JWT verification (Supabase HS256)
- Docker + Compose
- Render deploy config

## Quick Start

```bash
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env   # fill in your Supabase + Stripe keys
uvicorn app.main:app --reload
```

## Structure
```
app/
  main.py            FastAPI app (health, checkout, webhook, profile)
  config.py          Environment configuration
  supabase_client.py  Lazy Supabase client
  auth.py            Supabase JWT verification
  stripe_payments.py Stripe Checkout + webhook
tests/               Tests (pytest)
deploy/             Render blueprint
```

## Endpoints
- `GET /` — service info
- `GET /api/v1/health` — health check
- `POST /api/v1/checkout` — create a Stripe Checkout session (auth required)
- `POST /api/v1/webhook` — Stripe webhook receiver
- `GET /api/v1/profile` — current user (auth required)

## Notes
Supabase is used as the hosted backend, so no database container is required.
Set `SUPABASE_JWT_SECRET` to the value from your Supabase project settings so
`/api/v1/profile` and `/api/v1/checkout` can verify user tokens.
