## Quickstart
```bash
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env          # fill in SECRET_KEY + Stripe keys
python manage.py migrate
python manage.py runserver
```
Visit http://localhost:8000 — sign up, view `/payments/pricing/`, subscribe.

## Configure Stripe
1. Create products/prices in the Stripe dashboard and copy the **Price IDs** into
   `payments/views.py` (`PLANS`).
2. Set `STRIPE_SECRET_KEY`, `STRIPE_PUBLISHABLE_KEY`, `STRIPE_WEBHOOK_SECRET` in `.env`.
3. Expose the webhook locally: `stripe listen --forward-to localhost:8000/payments/webhook/`
   and copy the signing secret into `STRIPE_WEBHOOK_SECRET`.

## Deploy (Render)
- Push to GitHub and create a new **Blueprint** from `deploy/render.yaml`.
- Add `STRIPE_SECRET_KEY` in the Render dashboard env vars.
- `SECRET_KEY` and the database are provisioned automatically.
